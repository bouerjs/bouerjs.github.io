Using NPM

We recommended NPM installation method when building large scale applications with Bouer, working nicely with module bundlers like Webpack. 


    # latest stable version
    $ npm install bouerjs

Bouer also provides CLI project or components scaffolding, serve the application, builds and deploy.

Builds

There are differents of Bouer.js that we can find in dist/ folder, such as:

  • UMD – bouer.js can be used directly in the browser via a <script> tag. Minified version: bouer.min.js. (gzipped: 19.60kb).
  • ESM – bouer.esm.js stands for ES Modules. It is Javascript's proposal to implement a standard module system. Minified version: bouer.esm.min.js. (gzipped: 19.55kb).
  • CommonJS – bouer.common.js wraps each module in a function called require, and includes an object called module.exports, which exports code for availability to be required by other modules. Minified version: bouer.common.min.js. (gzipped: 19.59 kb).
