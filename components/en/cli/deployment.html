Deployment

The CLI allows you to build the bouer project to distribution, it has two options/modes one for development and the other for production. Follow the instructions bellow to build a bouer project:

Building: Development 


    bouer build

Building: Production 


    bouer build -m prod

You can also use --mix-config to combine configuration on build. All the built content, can be found in the dist folder of your project.
