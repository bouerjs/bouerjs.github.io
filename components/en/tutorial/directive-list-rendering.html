To be able to update the list, Bouer links with the main methods of an array, making them reactive.

Adding an item to the list can be like this: 


    app.data.list.push(/* ... */);

We can use .splice(...) or .filter(...) to remove items: 


    app.data.list.splice(index, qty);

We can also use filters like | where: or | order: to mutate the list before it's rendered. More details of e-for.
