Using the app.watch(/* ... */), we are able to watch for changes of a property: 


    app.watch('property', function(newValue, oldValue) {
      /* ... */
    })

By the default it targets the main application data, to target a specific one it needs to be defined as follow: 


    app.watch('property', function(newValue, oldValue) {
      /* ... */
    }, targetObject)

To be able unwatch a property, we use the destroy method inside of the returned object of watch method: 


    var watch = app.watch(/* ... */);

    // destroying the watch process
    watch.destroy();
